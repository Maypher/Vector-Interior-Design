Nodo admin Graphql API

Here are all the endpoints available to the admin backend of the Nodo website. It includes queries with access to private data and mutations to edit the data.

API Endpoints
https://127.168.0.8:8080/graphql

Queries

image

Description

Returns the image matching the given filename or null.

Response

Returns an Image

Arguments
Name Description
filename - String! The filename of the image to get.

Example

Query
query image($filename: String!) {
  image(filename: $filename) {
    id
    filename
    altText
    description
    descriptionFont
    sculpture
    index
    mainPage
    hideInProject
    phoneConfig {
      ...PhoneImageConfigFragment
    }
    desktopConfig {
      ...DesktopImageConfigFragment
    }
    space {
      ...SpaceFragment
    }
    mainImageConfig {
      ...MainPageImageConfigFragment
    }
    sculptureData {
      ...SculptureDataFragment
    }
  }
}

Mutations

createImage

Description

Creates a a new image for the given space. Returns an error if the space wasn't found or the filetype isn't supported.

Response

Returns an ImageSpaceNotFoundImageUnsupportedFileType!

Arguments
Name Description
spaceId - Int! The ID of the space this image will belong to.
image - Upload! The image file to upload.
altText - String! The alt text of the image. This is meant for impaired vision, not a description.

Example

Query
mutation createImage(
  $spaceId: Int!,
  $image: Upload!,
  $altText: String!
) {
  createImage(
    spaceId: $spaceId,
    image: $image,
    altText: $altText
  ) {
    ... on Image {
      ...ImageFragment
    }
    ... on SpaceNotFoundImage {
      ...SpaceNotFoundImageFragment
    }
    ... on UnsupportedFileType {
      ...UnsupportedFileTypeFragment
    }
  }
}
Variables
{
  "spaceId": 987,
  "image": Upload,
  "altText": "xyz789"
}
Response
{"data": {"createImage": Image}}

updateImage

Description

Updates the image identified by filename. All parameters are optional. Returns the updated image or null if no image was found.

Response

Returns an Image

Arguments
Name Description
filename - String! The filename of the image to update.
altText - String The new alt text of the image. Default = null
description - String The new description of the image. Default = null
descriptionFont - String The font used for the description. Default = null
index - Int The zero based index of where to position the image relative to all others. Default = null
mainPage - Boolean Set this image to be shown in the main page. Default = null
hideInProject - Boolean Hide this image in the project. Default = null
sculpture - Boolean Indicates that this image is a sculpture. Default = null
phoneConfig - PhoneConfigInput The configuration for the image display in phones. Default = null
desktopConfig - DesktopConfigInput The configuration for the image display in desktop. Default = null

Example

Query
mutation updateImage(
  $filename: String!,
  $altText: String,
  $description: String,
  $descriptionFont: String,
  $index: Int,

  $hideInProject: Boolean,

  $desktopConfig: DesktopConfigInput

  updateImage(
    filename: $filename,
    altText: $altText,

    mainPage

    space {

    }

      ...MainPageImageConfigFragment
    }

    }
  }

The configuration for an image that it's shown in the main page.

Fields
Field Name Description
id - Int! The id of this image config in the database.
descriptionEs - String The description of this image in Spanish.
descriptionEn - String The description of this image in English.
descriptionFont - String! The font of the description text.
descriptionFontSize - Float! The font size of the description text in rem units.
descriptionAlignment - String! The alignment of the description. Will be one of tailwind's text-align values.
index - Float! The index of this image in the main page in the form of a float.
phoneConfig - MainPageImagePhoneConfig! The configuration used to display on mobile devices.
desktopConfig - MainPageImageDesktopConfig! The configuration used to display on desktop devices.
image - Image! The image that owns this configuration.
Example
{
  "id": 987,
  "descriptionEs": "xyz789",
  "descriptionEn": "xyz789",
  "descriptionFont": "xyz789",
  "descriptionFontSize": 987.65,
  "descriptionAlignment": "abc123",
  "index": 987.65,
  "phoneConfig": MainPageImagePhoneConfig,
  "desktopConfig": MainPageImageDesktopConfig,
  "image": Image
}

MainPageImageDesktopConfig

Description

The configuration of how an image will look on the main page of desktop.

Fields
Field Name Description
imagePosition - DesktopImagePosition! The position of the image in the screen.
descriptionPosition - Location The position of the description relative to the image.
descriptionBorders - Borders! The borders of the description.
logoPosition - Location The position of the logo relative to the image.
logoBorders - Borders! The borders of the logo
descriptionLogoPosition - Location The position of the logo relative to the description.
descriptionLogoBorders - Borders! The borders of the description logo.
overflow - Boolean! Determines if the image should overflow to the borders of the screen.
Example
{
  "imagePosition": "LEFT",
  "descriptionPosition": "N",
  "descriptionBorders": Borders,
  "logoPosition": "N",
  "logoBorders": Borders,
  "descriptionLogoPosition": "N",
  "descriptionLogoBorders": Borders,
  "overflow": false
}

Project

Description

The main structure of the database.

Fields
Field Name Description
id - Int! The ID of the project in the database.
name - String! The name of the project.
description - String! The description of the project written in markdown format.
area - Int! The area of the project in meters squared.
index - Float! The index of the project for UI ordering purposes. It's a float due to how the database handles reordering.
public - Boolean! Indicates if the project is available in the public end of the website.
thumbnail - Image A project can have a thumbnail that will be the first thing shown in the UI.
spaces - [Space!]! All the spaces belonging to this project.
Example
{
  "id": 123,
  "name": "xyz789",
  "description": "xyz789",
  "area": 987,
  "index": 987.65,
  "public": false,
  "thumbnail": Image,
  "spaces": [Space]
}
